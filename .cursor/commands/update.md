# update

## Paper

Use this process to make updates to the paper.tex source file when prompted explicitly to update it based on changes that have accumulated to article.html (the source of truth) over recent commits:

- Determine the last commit in which the paper.tex file was changed
- Create a diff between article.html at that comment and the current commit
- Summarize these diffs in a TODO list of updates to make
- Ask clarifying questions before creating that list
- Review list with operator before continuing at operator's discretion
- Use the build script for the paper when recreating the PDF from source
